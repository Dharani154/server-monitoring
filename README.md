# Server Health Monitoring & Auto-Healing

## Project Overview

This project monitors Linux server health using Prometheus and Node Exporter. It visualizes metrics in Grafana, sends alerts through Alertmanager, and performs automatic recovery using Jenkins and Bash scripts.

## Tech Stack

- Linux
- Bash
- Git
- GitHub
- Jenkins
- Docker
- Kubernetes
- Prometheus
- Grafana
- Alertmanager
